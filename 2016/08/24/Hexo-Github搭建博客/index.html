Hexo+Github搭建博客(MAC)

  • github
  • hexo

一、准备工作

  • github(Desktop客户端即可)

  • homebrew

  • node.js

  • npm(安装node.js之后就可以了)

  • 环境为mac

二、安装node.js

安装node.js

两个版本均可，我遇到的一个问题是卡在验证安装包，别急重启就再次点击安装包安装

验证安装是否成功

终端指令
liangxianhuadeiMac:~ liangxianhua$ node -v
v4.5.0
liangxianhuadeiMac:~ liangxianhua$ npm -v
2.15.9

显示出版本号代表安装成功

三、安装homebrew

先检测下是否已安装
liangxianhuadeiMac:~ liangxianhua$ brew -v
Homebrew 0.9.9 (git revision cb70; last commit 2016-08-23)
Homebrew/homebrew-core (git revision bcb4a; last commit 2016-08-24)

如果显示版本号代表已经安装过直接看第四步

打开终端粘贴脚本
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

安装完成验证是否成功如果有其他问题看homebrew官网安装方式

四、安装hexo

安装
npm install -g hexo

过程可能较慢根据网络情况(最好是科学上网😆你懂得)

五、Github设置

1.桌面创建一个文件夹，文件名自定义，打开文件夹新建一个名为hexo文件夹，hexo文件里再新建一个名为public文件夹，此public文件夹作为仓库

2.利用客户端新建一个名字叫你的github用户名.github.io的项目，然后推到github网页

3.如果你有自己申请的顶级域名，并想链接到你的网页，你还需要在public创建一个没有后缀的名为CANME的文件，用notepad打开，输入你的域名，比如“xhliang.com”

六、hexo写文章

终端打开你的hexo文件夹执行
hexo init
mpm install

接着hexo会自动生成所需要的所有文件

等待执行完毕

hexo s

通过浏览器打开“http://localhost:4000/”

假如没有正常显示换Safari浏览器尝试

新建文章

比如新建一个名为Hexo的文章

liangxianhuadeiMac:hexo liangxianhua$ hexo new "Hexo"

七、提交到github

将public文件里的内容提交到github上

你的博客地址就是“用户名/用户.github.io”

切记项目名称必须是上诉地址格式

最后打开你的博客地址，假如你是购买过域名并且在第五步github设置中输入自己的域名，则可以直接打开你域名地址，域名购买域名解析就需要你自己选择啦😆。这里就推荐个别人给我推荐的:去他爹
